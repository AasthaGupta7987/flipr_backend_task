# E-commerce Backend with FastAPI and MySQL

## Overview
This project implements a complete backend for an e-commerce platform using FastAPI and MySQL, including:
- **User Authentication**
- **Product Management**
- **Cart Management**
- **Order Management**

## Prerequisites
- Python 3.8+
- MySQL server
